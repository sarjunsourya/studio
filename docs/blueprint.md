# **App Name**: The Divine Kitchen

## Core Features:

- Luxurious Home Page: Display a high-quality hero image and featured dishes to convey premium quality. The hero image should include the logo and tagline 'Home-Cooked Goodness by Roopa Gokul in Delft' and the sub-tagline 'Authentic homemade meals, crafted fresh with love'.
- Dynamic Weekly Menu: A page displaying a weekly menu list with filters for dietary preferences using glassmorphism cards. Include an optional 'Order Inquiry' button under each dish.
- Catering Inquiry Form: Enable users to submit catering inquiries with a form, capturing event details and preferences. Fields should include: Name, Event Date, Number of Guests, Menu Preferences, and Message.
- Image Gallery: Display high-quality food photography in a gallery with a masonry-style layout, including "Behind the Scenes" content. Integrate with Instagram if possible, otherwise use placeholders.
- Contact Form and Map: Include a contact form for inquiries and a map displaying the Delft location (Raad van Europalaan 62, 2625PC Delft). Also add buttons for WhatsApp message and Instagram link (https://instagram.com/thedivinekitchen5/). Include contact email: info@the-divine-kitchen.com

## Style Guidelines:

- Primary color: Gold (#F3C654) for warmth and luxury, inspired by homemade kitchen themes.
- Background color: Very light grey (#F5F5F5) for a clean and modern look.
- Accent color: Green (#3FAE49) to complement the gold, creating balance, contrast and freshness.
- Headline font: 'Poppins' sans-serif font for a modern and luxurious feel. Note: currently only Google Fonts are supported.
- Body font: 'Inter' sans-serif font for readability and a clean aesthetic. Note: currently only Google Fonts are supported.
- Use simple and elegant icons in brand colors for 'Why The Divine Kitchen?' section. Icons should represent: Homemade, Fresh Ingredients, Hygienic Preparation, Authentic Flavors, and Delft-based home kitchen.
- Implement glassmorphism across cards, forms, and highlights with translucent backgrounds (rgba(255, 255, 255, 0.25)), soft shadows (8-12px blur), rounded corners (20-28px), and light, glowing borders (1px solid rgba(255, 255, 255, 0.45)). Use backdrop-filter: blur(14px).
- Incorporate subtle animations such as fade-in effects, hover glows, and smooth transitions (200-300ms). Consider a parallax hero image if possible.
